# Markdown to Google Docs Converter

This project converts structured markdown meeting notes into a fully formatted Google Doc using the Google Docs API.

## Features

- Heading hierarchy (H1 / H2 / H3)
- Nested bullet lists
- Google Docs checkboxes
- Styled @mentions
- Footer styling
- Runs directly in Google Colab

## Setup

1. Open the notebook in Google Colab
2. Run all cells
3. Authenticate with your Google account
4. A new Google Doc will be created automatically

## Dependencies

- google-api-python-client
- google-auth
- markdown

## Running in Colab

Simply open the notebook and execute cells top-to-bottom.

## Output

A new Google Doc containing the formatted meeting notes will appear in your Google Drive.
